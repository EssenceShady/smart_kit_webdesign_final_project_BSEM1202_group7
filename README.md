# SMART KIT

## Learn Today, Lead Tomorrow

Smart Kit is an educational web platform developed to provide students with easy access to learning resources, examination information, educational updates, and past examination papers.

The project was developed as part of the **BSEM1202 Group Project**.

---

## Project Overview

Smart Kit aims to support students by creating a centralized platform where educational resources can be easily accessed.

The platform currently provides:

* Educational News & Updates
* Downloadable Past Examination Papers
* About Us Information
* Contact Information
* Responsive User Interface

Future versions of the platform will include additional features such as online book sales, result checking services, student accounts, and scholarship opportunities.
